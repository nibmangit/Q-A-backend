# Full Stack Q&A Platform — Backend

Backend API built with **Django + Django REST Framework** for a full-stack Q&A platform. Handles authentication, questions, answers, discussion rooms, notifications, and more.

## 🚀 Features
- **Authentication**: JWT-based login, Google OAuth
- **Password Management**: Password reset via secure email with token verification
- **Questions & Answers**: CRUD endpoints for questions and answers
- **Discussion Rooms**: Each question has a chat room with role-based permissions
- **Engagement System**: Likes/dislikes, points, badges
- **Notifications**: Fetch user activity notifications
- **Admin Features**: Role-based permissions for managing rooms

## 🛠️ Tech Stack
- Django, Django REST Framework
- SQLite (development) / PostgreSQL (production ready)
- Email integration via SMTP
- JWT & OAuth for authentication
